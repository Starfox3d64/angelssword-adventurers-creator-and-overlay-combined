Place your character assets here.

Supported formats: .webm, .webp, .gif, .png, .mp4

Required files:
  neutral_idle.<ext>       Default resting state
  neutral_speaking.<ext>   Talking, neutral expression
  happy_idle.<ext>         Smiling, not talking
  happy_speaking.<ext>     Smiling and talking
  sad_idle.<ext>           Frowning, not talking
  sad_speaking.<ext>       Frowning and talking
  eyes_closed.<ext>        Eyes closed for 1.5+ seconds

If no assets are found, the app generates colored placeholders automatically.
