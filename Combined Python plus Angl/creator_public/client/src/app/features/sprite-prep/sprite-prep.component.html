ℹ️ About Manual Upload
Bring your own character art into the pipeline as a chroma-keyed sprite. The image is placed on a 1280×720 canvas so later video and export steps share one frame size.
Best source images:
  • 16:9 aspect — matches the pipeline canvas
  • ~720p (1280×720) — sharp enough without huge files
  • Clear contrasted color behind the character (green, magenta, etc.) so chroma key can cut cleanly
  • Headroom and side room — leave empty space above and beside the figure so advanced emote animations can move without clipping the frame
Frame editor: drag to move, corners to rotate, zoom slider (100% = original size). Empty areas fill with your key color. Download / handoff only save the clipped 1280×720 frame.
Key color: on upload we try to auto-pick the background key. Choosing a swatch yourself overrides that pick and persists through the rest of the workflow (video gen, prep, and export).
💡 No API key required for Manual Upload. Prefer AI Generate if you want a model to paint the sprite for you.
